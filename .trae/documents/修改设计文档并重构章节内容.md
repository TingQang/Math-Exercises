# 修改设计文档并重构章节内容

## 1. 修改设计文档
- 删除篇章结构中的subsection层级，将题型直接放在section下
- 修改整体结构示意图，移除subsection相关内容
- 更新章节内部结构描述，删除subsection相关说明
- 修改题型归类描述，将题型直接关联到section层级

## 2. 清理项目章节文件
- 删除content目录下所有现有的.tex章节文件（ch00.tex到ch21.tex，ch99.tex）
- 保留必要的配置文件和主文件

## 3. 填充章节内容
- 根据数一考点题型总结.md的内容，创建新的章节.tex文件
- 在每章开头用无序列表总结考点
- 按照section-题型的结构组织内容
- 每个题型使用\pt命令标注
- 不需要例题，测试章节暂时不需要题目

## 4. 章节结构示例
```latex
\chapter{函数、极限、连续}

\section{考点说明}
- 函数的概念及表示法
- 函数的有界性、单调性、周期性和奇偶性
- 复合函数、反函数、分段函数和隐函数
- 基本初等函数的性质及其图形
- 初等函数

\section{函数}
\pt{复合函数解析式求解}
\pt{函数奇偶性、周期性、有界性判定}
\pt{分段函数定义域与值域求解}
\pt{函数性质的综合应用}
\pt{应用问题的函数关系建立}

\section{极限}
\pt{极限的概念性问题}
\pt{利用四则运算、重要极限求极限}
\pt{利用夹逼准则、单调有界准则求极限}
\pt{无穷小量阶的比较}
\pt{极限式中参数的确定}
\pt{无穷大量与无界变量的区分}

\section{连续}
\pt{函数连续性的判定}
\pt{函数间断点的识别与类型判断}
\pt{闭区间上连续函数性质的应用}
\pt{利用连续性求极限}
```

## 5. 注意事项
- 严格按照设计文档的篇章结构填充内容
- 每章开头必须有考点总结（无序列表）
- 不需要例题，只保留题型和考点
- 测试章节暂时不需要题目
- 确保文件命名符合设计文档规范
