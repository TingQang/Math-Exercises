\chapter*{附录}

\section{常用公式}

\subsection{高等数学}

1. **极限公式**：
   - $\lim_{x \to 0} \frac{\sin x}{x} = 1$
   - $\lim_{x \to \infty} \left(1 + \frac{1}{x}\right)^x = e$

2. **导数公式**：
   - $(x^n)' = nx^{n-1}$
   - $(\sin x)' = \cos x$
   - $(\cos x)' = -\sin x$
   - $(e^x)' = e^x$
   - $(\ln x)' = \frac{1}{x}$

3. **积分公式**：
   - $\int x^n dx = \frac{x^{n+1}}{n+1} + C$ (n ≠ -1)
   - $\int \sin x dx = -\cos x + C$
   - $\int \cos x dx = \sin x + C$
   - $\int e^x dx = e^x + C$
   - $\int \frac{1}{x} dx = \ln|x| + C$

\subsection{线性代数}

1. **行列式性质**：
   - 转置行列式的值不变
   - 交换两行（列），行列式变号
   - 行列式某一行（列）的公因子可以提到行列式符号外面

2. **矩阵运算**：
   - $(AB)^T = B^T A^T$
   - $(A^{-1})^T = (A^T)^{-1}$
   - $(kA)^{-1} = \frac{1}{k} A^{-1}$ (k ≠ 0)

\subsection{概率论与数理统计}

1. **概率公式**：
   - 加法公式：$P(A \cup B) = P(A) + P(B) - P(A \cap B)$
   - 乘法公式：$P(AB) = P(A)P(B|A)$
   - 全概率公式：$P(A) = \sum_{i=1}^n P(B_i)P(A|B_i)$
   - 贝叶斯公式：$P(B_j|A) = \frac{P(B_j)P(A|B_j)}{\sum_{i=1}^n P(B_i)P(A|B_i)}$

2. **常见分布的期望和方差**：
   - 二项分布 $B(n,p)$：$E(X) = np$，$D(X) = np(1-p)$
   - 泊松分布 $P(\lambda)$：$E(X) = \lambda$，$D(X) = \lambda$
   - 正态分布 $N(\mu, \sigma^2)$：$E(X) = \mu$，$D(X) = \sigma^2$
