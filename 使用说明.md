# 考研数学一习题集与错题集LaTeX笔记使用说明

## 1. 项目概述

本项目是一个基于LaTeX的考研数学一习题集与错题集管理系统，采用模块化设计，使用成熟的exercise宏包实现习题和答案的组织管理，结合hyperref宏包实现双向超链接功能。项目支持"章节-题型-习题"的三级结构，实现题目按知识点精准归类、难度分级标注、来源追溯以及错题专项复盘。

### 1.1 核心功能

- **章节-题型-习题结构**：清晰的三级组织方式，便于按知识点管理习题
- **双向超链接**：题目与答案之间可相互跳转
- **难度可视化**：1-5级难度，彩色方块直观显示
- **习题信息标注**：支持来源、考点、提示等信息
- **答案详细解析**：支持易错点、一题多解等功能
- **矢量图支持**：支持插入矢量图，实现图文结合的题目
- **模块化设计**：配置与内容分离，便于维护和扩展

## 2. 环境搭建

### 2.1 软件需求

- **TeX发行版**：MiKTeX（Windows）或TeX Live（Linux/macOS）
- **编译引擎**：XeLaTeX（推荐，支持中文）
- **编辑器**：VS Code + LaTeX Workshop插件（推荐）或其他LaTeX编辑器
- **Git**：版本控制

### 2.2 VS Code配置

项目已配置好`.vscode/settings.json`，支持自动编译和实时预览：

- 自动编译：保存文件时自动编译
- PDF预览：内置标签页预览，支持双击同步定位
- 输出目录：所有编译产物输出到`./build`目录
- 文件清理：自动清理编译中间文件

## 4. 章节组织规范

### 4.1 章节命名规则

- `ch00.tex`：前言
- `ch01.tex`-`ch10.tex`：正文章节（包含题目和答案）
- `ch99.tex`：附录

### 4.2 章节内部结构

```latex
\chapter{极限与连续}
\label{ch:limit-continuity}

\begin{exam-point}{极限的定义与性质}{limit-def-prop}
考点说明
\end{exam-point}

\section{题型1：极限的四则运算法则}

\begin{Exercise}[title={极限的四则运算法则}, label={ex:ch01-01}]
    \Question \difficulty{1} 计算 $\lim_{x \to 0} \frac{x^2 - 1}{x - 1}$ \exercisesource{考研数学一·2018}
    \matchexampoint{极限的四则运算法则}
    \exercisetip{直接代入即可}
\end{Exercise}

% 其他题型...

\section{本章答案与解析}

\begin{Answer}[ref={ex:ch01-01}]
    \Question 
    \textbf{易错点：} 直接代入会得到0/0型不定式，不能直接计算
    
    \multiplesolution{解法一：泰勒展开法}
    解答内容
    
    \multiplesolution{解法二：洛必达法则}
    解答内容
\end{Answer}
```

## 5. 习题和答案编写规范

### 5.1 习题编写

#### 5.1.1 Exercise环境

```latex
\begin{Exercise}[title={题型名称}, label={ex:chXX-YY}]
    \Question \difficulty{难度级} 题目内容 \exercisesource{来源}
    \matchexampoint{考查知识点}
    \exercisetip{解题提示}
    
    % 更多题目...
\end{Exercise}
```

#### 5.1.2 习题信息标注

- `\difficulty{1-5}`：难度等级，1-5级，彩色方块显示
- `\exercisesource{来源}`：习题来源，如"考研数学一·2018"
- `\matchexampoint{考点}`：匹配的考点
- `\exercisetip{提示}`：解题提示

### 5.2 答案编写

#### 5.2.1 Answer环境

```latex
\begin{Answer}[ref={ex:chXX-YY}]
    \Question 
    \textbf{易错点：} 易错点说明
    
    \multiplesolution{解法一：方法名称}
    解答内容
    
    \multiplesolution{解法二：方法名称}
    解答内容
    
    % 更多题目答案...
\end{Answer}
```

#### 5.2.2 答案信息标注

- `\textbf{易错点：}`：易错点说明
- `\multiplesolution{解法名称}`：一题多解标注

### 5.3 矢量图插入

```latex
\begin{figure}[h]
    \centering
    \includegraphics[width=0.8\textwidth]{images/function_comparison.pdf}
    \caption{函数图像比较}
    \label{fig:function_comparison}
\end{figure}
```

## 6. 编译和构建

### 6.1 使用latexmk自动编译（推荐）

```bash
latexmk -xelatex -synctex=1 -interaction=nonstopmode -file-line-error -shell-escape -outdir=./build main.tex
```

### 6.2 使用VS Code自动编译

1. 打开项目文件夹
2. 安装LaTeX Workshop插件
3. 打开任意`.tex`文件
4. 保存文件时自动编译
5. 在右侧标签页查看PDF预览

### 6.3 手动编译流程

```bash
xelatex -synctex=1 -interaction=nonstopmode -file-line-error -shell-escape -output-directory=./build main.tex
# 如有参考文献
# biber ./build/main

xelatex -synctex=1 -interaction=nonstopmode -file-line-error -shell-escape -output-directory=./build main.tex
```

## 8. 高级功能使用

### 8.1 自定义命令

项目定义了丰富的自定义命令，便于数学符号的快捷输入：

- **集合符号**：`\R`, `\N`, `\Z`, `\Q`, `\C`
- **微分与导数**：`\ddme`, `\pd`, `\pder`, `\gradme`
- **极限与积分**：`\limn`, `\limx`, `\limxa`, `\intab`
- **矩阵与向量**：`\mat`, `\bmat`, `\absme`, `\normme`
- **概率与统计**：`\P`, `\E`, `\Var`, `\Cov`

### 8.2 自定义环境

- `exam-point`：考点环境
- `Exercise`：习题集合环境
- `Answer`：答案环境

### 8.3 错题集使用


## 9. 项目扩展

### 9.1 新增章节

1. 在`content/`目录创建新的`.tex`文件，如`ch11.tex`
2. 在`main.tex`中添加`\input{content/ch11}`语句
3. 按照章节组织规范编写内容

### 9.2 新增题型

1. 在章节文件中添加新的`\subsection{题型名称}`
2. 使用`Exercise`环境编写习题
3. 在答案部分添加对应的`Answer`环境

### 9.3 自定义配置

- **添加宏包**：在`config/packages.tex`中添加
- **定义颜色**：在`config/colors.tex`中添加
- **定义样式**：在`config/styles.tex`中添加
- **定义命令**：在`config/commands.tex`中添加
- **定义环境**：在`config/environments.tex`中添加

## 11. 常见问题与解决方案

### 11.1 编译错误

- **缺少宏包**：使用包管理器安装缺失的宏包
- **语法错误**：检查LaTeX语法，特别是环境的开始和结束
- **路径错误**：检查图片路径是否正确

### 11.2 中文显示问题

- 确保使用XeLaTeX编译引擎
- 确保已加载ctex宏包

### 11.3 超链接问题

- 确保已加载hyperref宏包
- 检查标签名称是否唯一
- 确保标签引用正确
