# 考研数学一习题集与错题集LaTeX笔记使用说明

## 1. 项目简介

本项目是一个基于LaTeX的考研数学一习题集与错题集管理系统，采用模块化设计，支持题目按知识点精准归类、难度分级标注、来源追溯以及错题专项复盘。

### 1.1 核心功能

- **四级结构化组织**: 章节-小节-题型-习题
- **难度可视化系统**: 1-5级难度，彩色方块直观展示
- **集中式答案组织**: 每章末尾集中给出答案与解析
- **超链接跳转**: 答案与题目间可相互跳转
- **矢量图支持**: 支持嵌入高质量矢量图
- **错题分析系统**: 支持错题原因分析和复盘管理

## 2. 环境准备

### 2.1 软件安装

1. **TeX发行版**: 安装MiKTeX (Windows)或TeX Live (Linux/macOS)
2. **编辑器**: 推荐使用VS Code + LaTeX Workshop插件
3. **版本控制**: Git (可选)

### 2.2 VS Code配置

项目已包含`.vscode/settings.json`配置文件，自动配置LaTeX Workshop插件，支持:
- 自动编译
- 反向搜索
- PDF预览

## 3. 项目结构

本项目采用模块化设计，将配置、内容和资源分离，便于维护和扩展：

```
├── config/                  # 配置文件目录
│   ├── colors.tex           # 颜色配置
│   ├── environments.tex     # 环境定义
│   ├── commands.tex         # 自定义命令
│   ├── packages.tex         # 宏包加载
│   └── styles.tex           # 样式配置
├── content/                 # 内容文件目录（存放题目）
│   ├── ch00.tex             # 前言
│   ├── ch01.tex             # 第1章：极限与连续
│   ├── ch02.tex - ch10.tex  # 正文章节
│   └── ch99.tex             # 附录
├── answers/                 # 答案文件目录（存放答案）
│   ├── ans01.tex            # 第1章答案
│   ├── ans02.tex - ans10.tex  # 其他章节答案
│   └── ans99.tex            # 附录答案
├── figures/                 # 图片资源
│   ├── common/              # 通用图片
│   └── ch01/                # 第1章图片
├── images/                  # 矢量图资源
├── scripts/                 # 脚本目录
├── main.tex                 # 主文档入口
├── references.bib           # 参考文献
├── build/                   # 编译输出目录
└── 设计文档.md              # 设计文档
```

## 4. 快速开始

### 4.1 编译文档

#### 方法一：使用latexmk自动编译（推荐）

```bash
latexmk -xelatex -synctex=1 -interaction=nonstopmode -file-line-error -shell-escape -outdir=./build main.tex
```

#### 方法二：使用脚本编译

```bash
# Windows
scripts/build.bat

# Linux/macOS
bash scripts/build.sh
```

#### 方法三：VS Code自动编译

1. 打开项目文件夹
2. 打开`main.tex`文件
3. 保存文件，LaTeX Workshop会自动编译
4. 在右侧预览PDF

### 4.2 查看输出

编译成功后，PDF文件将生成在`build/main.pdf`路径下。

## 5. 章节编写规范

### 5.1 章节结构

每个章节采用以下结构：

```latex
\chapter{章节名称}
\label{ch:label}

\section{小节名称}

\begin{exam-point}{考点名称}{标签}
考点说明
\end{exam-point}

\subsection{题型1 题型名称}
\begin{enumerate}[label={【\arabic*】}]
    \item \label{ex:chXX-TY-MM} \difficulty{难度级} 题目内容 \exercisesource{来源}
    \matchexampoint{考点}
    \exercisetip{提示}
\end{enumerate}
```

### 5.2 答案文件结构

答案文件单独存放在`answers/`目录下，命名为`ansXX.tex`，结构如下：

```latex
\chapter{第1章答案与解析}
\label{ch:ch01-answers}

\begin{chapter-answers}
    \begin{type-answers}{题型1 题型名称}
        \begin{answer-item}{ex:chXX-TY-01}
            答案内容
        \end{answer-item}
    \end{type-answers}
\end{chapter-answers}
```

### 5.3 习题编写

#### 5.3.1 基本结构

```latex
\item \label{ex:ch01-01-01} \difficulty{1} 计算 $\lim_{x \to 0} \frac{x^2 - 1}{x - 1}$ \exercisesource{考研数学一·2018}
\matchexampoint{极限的四则运算法则}
\exercisetip{直接代入即可}
```

#### 5.3.2 难度标注

使用`\difficulty{等级}`命令标注难度（1-5级）：

- 难度1：绿色（基础题）
- 难度2：蓝色（简单题）
- 难度3：橙色（中等题）
- 难度4：红色（难题）
- 难度5：紫色（极难题）

#### 5.3.3 矢量图插入

```latex
\item \label{ex:ch01-10-01} \difficulty{3} 观察下图中函数 $y = \sin(x)$ 与 $y = x - \frac{x^3}{6}$ 的图像...
\begin{figure}[ht]
    \centering
    \includegraphics[width=0.8\textwidth]{images/function_comparison.pdf}
    \caption{函数图像比较}
    \label{fig:function_comparison}
\end{figure}
```

### 5.4 答案编写

使用`chapter-answers`、`type-answers`和`answer-item`环境编写答案，答案文件单独存放在`answers/`目录下：

```latex
\chapter{第1章答案与解析}
\label{ch:ch01-answers}

\begin{chapter-answers}
    \begin{type-answers}{题型1 极限的四则运算法则}
        \begin{answer-item}{ex:ch01-01-01}
            $\lim_{x \to 0} \frac{x^2 - 1}{x - 1} = \lim_{x \to 0} (x + 1) = 1$（直接代入）
        \end{answer-item}
    \end{type-answers}
\end{chapter-answers}
```

## 6. 自定义配置

### 6.1 修改颜色

编辑`config/colors.tex`文件，修改或添加颜色定义：

```latex
\definecolor{kaoyan-blue}{RGB}{0, 102, 204}    % 主色调蓝色
\definecolor{difficulty-1}{RGB}{0, 153, 0}      % 难度1：绿色
```

### 6.2 添加自定义命令

编辑`config/commands.tex`文件，添加自定义命令：

```latex
\newcommand{\mycommand}[1]{...}
```

### 6.3 添加自定义环境

编辑`config/environments.tex`文件，添加自定义环境：

```latex
\newenvironment{myenvironment}[1]{...}{...}
```

### 6.4 修改样式

编辑`config/styles.tex`文件，修改页面布局、标题格式等：

```latex
\geometry{
    a4paper,
    left=3cm,
    right=2.5cm,
    top=2.5cm,
    bottom=2.5cm
}
```

## 7. 错题集使用

### 7.1 错题结构

使用`error-analysis`环境记录错题：

```latex
\begin{error-analysis}
\textbf{错因：} 错误原因分析

\textbf{错误解法：}
原错误解题过程

\textbf{正确解法：}
正确解题过程

\textbf{复盘状态：} 未复盘/已复盘N次/已掌握
\end{error-analysis}
```

### 7.2 组织方式

- 错题按原章节归类，放入对应的章节文件中
- 使用`\ref{}`命令交叉引用到原题
- 定期更新复盘状态

## 8. 扩展与维护

### 8.1 添加新章节

1. 在`content/chapters/`目录下创建新的`.tex`文件
2. 按章节编写规范编写内容
3. 在`main.tex`中添加`\input{content/chapters/新章节.tex}`

### 8.2 添加新题型

1. 在章节文件中使用`\subsection{题型名称}`创建新题型
2. 使用`enumerate`环境添加习题
3. 在答案部分添加对应的答案

### 8.3 性能优化

- 图片压缩：使用矢量图或压缩后的位图
- 代码清理：定期清理无用的宏包和命令
- 编译缓存：使用latexmk自动管理缓存

## 9. 常见问题

### 9.1 编译错误

- **缺少宏包**: 安装缺失的宏包
- **语法错误**: 检查LaTeX语法，特别是括号匹配
- **图片路径错误**: 确保图片路径正确
- **编码问题**: 确保使用UTF-8编码

### 9.2 预览问题

- **PDF无法预览**: 检查编译是否成功，查看日志文件
- **反向搜索失效**: 确保VS Code和PDF阅读器正确配置

### 9.3 格式问题

- **样式不一致**: 检查样式配置文件
- **页码错误**: 检查章节结构和页码设置

## 10. 快捷键

- **Ctrl + Alt + J**: 正向搜索（从LaTeX到PDF）
- **Ctrl + Click**: 反向搜索（从PDF到LaTeX）
